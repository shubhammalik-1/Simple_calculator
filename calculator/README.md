 Web Calculator

Simple client-side calculator built with HTML, CSS and JavaScript.

Usage

- Open [index.html](index.html) in a browser.
- Use mouse/touch to click buttons or type on your keyboard.

Features

- Basic arithmetic: `+ - * /`
- Percent (`%`)
- Parentheses and decimals
- Keyboard support, backspace, clear, Enter to evaluate

Files

- [index.html](index.html) — UI
- [style.css](style.css) — styles
- [script.js](script.js) — logic and keyboard handling
